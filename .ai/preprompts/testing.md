# Testing Mindset

Before finishing any feature, think about:

- What happens if the player clicks the wrong thing?
- What happens near the edge of the map?
- What happens if no valid tile exists?
- What happens if a city has no free space?
- What happens if the player ends turn repeatedly?
- What happens if multiple systems interact?

Always provide:
- Manual test steps
- Expected result
- Edge cases
- Possible exploit paths
