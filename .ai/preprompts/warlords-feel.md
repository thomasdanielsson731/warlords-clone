# Warlords Feel

Always preserve the emotional feeling of Warlords II:

- One more turn
- Tension while expanding
- Satisfaction when capturing cities
- Excitement when exploring ruins
- Memorable heroes
- Rivalry between factions
- Strategic map control
- Large fantasy war

The game should create:
- Curiosity
- Triumph
- Risk
- Attachment
- Surprise

If a feature is technically correct but does not create these feelings, improve it.
